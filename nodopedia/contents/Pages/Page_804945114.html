Position → Angle Nodon


Turning positions into angles is my whole
reason for being.


Port

Input

Port name Port function Input operation
Horizontal position The value received here is used as the
horizontal position when calculating
the output.		 Signal passes through unchanged
Vertical position The value received here is used as the vertical position when calculating the output. Signal passes through unchanged

Output

Port name Port function Output value
Angle

The two inputs together specify a location. This Nodon will find the angle of the direction along which that location lies, from the center (0,0), then send it as output. Measured in degrees (°).

 Between -180.00 and 180.00
