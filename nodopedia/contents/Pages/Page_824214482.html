Port

Input

Port name Port function Input operation
Slowdown amount

Slows down time according to the input value received at this port.
The larger the value, the stronger the effect of the slowdown. A value of 1.00 will slow time down to one thousandth of the normal speed, practically stopping it altogether.
The slowdown only affects the movement of objects. Other operations such as calculations, text and number displays, and Timer Nodon actions will work as usual.

 Restricted to between 0.00 and 1.00
