Angle-Difference Nodon


Ah, I know that look! You're wanting to calculate
some angles, aren't ya? Instead of subtracting
one from the other, you might want to give
Angle Difference a try!


Port

Input

Port name Port function Input operation
Angle 1

Takes the input value as the reference
angle, measured in degrees (°).

 Signal passes through unchanged
Angle 2

Takes the input value as the angle to be subtracted.
Measured in degrees (°).

 Signal passes through unchanged

Output

Port name Port function Output value
Difference

Outputs the difference between angle 1
and angle 2.
This isn't a simple subtraction, but a calculation of the difference only within a single lap of a circle.
For example, inputting 10.00 and 380.00 results in not -370.00, but -10.00.
Measured in degrees (°).

 Between -180.00 and 180.00
