Vibration Nodon


Vrrrrrrrrr, I make
the Joy-Con controller
vibraaaaate.


Port

Input

Port name Port function Input operation
Vibration strength

If a value greater than 0.00 is received at this port, the Joy-Con will vibrate.
The greater the value, the stronger the vibrations.

Restricted to between 0.00 and 1.00

Output

None

Connection

None

Settings

Controller Number

Sets the number of the controller that will vibrate.

Setting value Description
Between 1 and 4 Causes the Joy-Con with the specified number to vibrate.
 Makes the Joy-Con controllers attached to the Nintendo Switch console vibrate.
This feature is not available on Nintendo Switch Lite.

Which Controller?

Sets a controller to vibrate, either Joy-Con (R) or Joy-Con (L).

Continue Output for How Long?

Sets the number of seconds that the output will continue for.
Can be set to between 0.00 and 100.00.

Frequency

Sets the frequency of the vibration.
Can be set between 40 and 1280. Measured in hertz (cycles per second).

Other

Because the vibration feature is not available on the Nintendo Switch Lite system, this Nodon cannot be used. The Nodon will become available if a Joy-Con controller is wirelessly connected to the Nintendo Switch Lite system.
