Camera-Direction Nodon


Yeah, swing the camera!
This way! Now that way!


Port

Input

Port name Port function Input operation
Up/Down Moves the camera's direction up or down depending on the value received as input.
Positive values will move it up, negative values down.
Measured in degrees (°).		 Loops between -180.00 and 180.00
Horizontal Moves the camera's direction horizontally depending on the value received as input.
Positive values will move it right, negative values left.
Measured in degrees (°).		 Loops between -180.00 and 180.00

Output

None

Connection

None

Settings

None

Other

You can only call up one Camera-Direction Nodon per game.
Also, you can't call up a Game-Screen Nodon or a Camera Nodon at the same time as the Camera-Direction Nodon.
