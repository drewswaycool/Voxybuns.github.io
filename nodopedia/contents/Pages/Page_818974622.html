Wormhole-Entrance Nodon


This is Entrance. Input value will be transmitted
to Exit on schedule, five by five, over!


Port

Input

Port name Port function Input operation
Entrance

Transmits the received value to a Wormhole-Exit Nodon with the same Wormhole ID.
If there are multiple Wormhole-Entrance Nodon with the same ID, the value of the signals will be added together and transmitted.

 Signal passes through unchanged
