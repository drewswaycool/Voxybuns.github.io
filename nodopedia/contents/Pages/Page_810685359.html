Port

Input

Port name Port function Input operation
Angle The input received at this port will be used as an angle to calculate the location.
Measured in degrees (°).		 Loops between -180.00 and 180.00

Output

Port name Port function Output value
Horizontal position

Using the value received at the angle port, this calculates the location of the angle on a circle with a radius of 1.00, then outputs the horizontal location.
This will be the cos (cosine) of the angle.

 Between -1.00 and 1.00
Vertical position

Using the value received at the angle port, this calculates the location of the angle on a circle with a radius of 1.00, then outputs the vertical location.
This will be the sin (sine) of the angle.

 Between -1.00 and 1.00
