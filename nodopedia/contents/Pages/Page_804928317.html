Object-Break Nodon


Hey, hon! I'll look out for those broken objects!
Don't you worry!


Port

Input

None

Output

Port name Port function Output value
Broken count

This port outputs the number of applicable objects that break in the game's world, at the moment that they break.

 Accepts any number

Connection

None

Settings

Check What?

Sets the type of objects that will be checked.
You can enable multiple types.

If multiple object types are enabled, the Nodon will check all of them and output the total number of breakages.
